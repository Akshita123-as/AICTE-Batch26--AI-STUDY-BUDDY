# AICTE-Batch26--AI-STUDY-BUDDY
AI Study Buddy is an AI-powered learning assistant that helps students study smarter by automatically summarizing notes, answering questions, generating quizzes, and creating flashcards from uploaded PDF documents.
The application is built using Python, Streamlit, Google Gemini AI, PyPDF, and OCR (Tesseract) to support both text-based and scanned PDF documents.

# 🚀 Features:
📖 Smart Notes Summarization
* Upload study notes in PDF format.
* Generates concise bullet-point summaries.
* Converts lengthy content into easy-to-understand key points.
# ❓ AI Question Answering: 
Ask questions related to uploaded notes.
Answers from the uploaded notes whenever possible.
Uses AI knowledge when information is not available in the notes.
# 📝 Quiz Generator:
Automatically generates multiple-choice questions (MCQs).
Includes 4 options for each question.
Displays correct answers for self-assessment.
# 🗂 Flashcard Generator:
Creates flashcards from study materials.
Question and answer format for quick revision.
Helps improve memory retention.
# 📄 PDF Text Extraction:
Reads normal text-based PDFs.
Extracts content using PyPDF.
# 🖼 OCR for Scanned PDFs:
Supports scanned documents and image-based PDFs.
Uses Tesseract OCR to convert images into text.
Allows AI processing of scanned notes.
# 🛠 Technologies Used:
Python
Streamlit
Google Gemini AI
PyPDF
PDF2Image
